---
description: Fazer commits individuais por arquivo na ordem cronológica
---

Preciso fazer commits individuais para ada um dos arquivos que foram alterados na ordem do mais antigo alterado para o mais novo.

- verifique com git status o que foi alterado
- ordene os arquivos do que foi alterado por último para o primeiro
- seguindo a ordem faça git add para o primeiro arquivo da fila
- use legal-commit para fazer o commit
- repita a operação até terminar todos os commits
- apresente um relatório do que foi feito
